# Naive-Bayesian-using-Java
Program penentuan keputusan menggunakan pengklasifikasian data metode Naive Bayes dengan sample data dinamis pada file bayesian.csv

  Cara menambahkan pengecekan data:
1. Tambahkan variabel pengukuran data pada file bayesian.csv
2. Ubah ukuran array pada method readFile menyesuaikan dengan jumlah data yang baru
3. Tambahkan scanner untuk dapat membaca data baru apabila data baru menambahkan kolom baru
